Помощь

Создание комнаты:
- Нажмите на объект в палитре (слева/сверху), чтобы он появился в центре камеры.
- Или перетащите его мышью на поле (Drag & Drop).
- Зажмите ЛКМ (или один палец на мобилке) на пустом поле камеры, чтобы перемещаться по комнате.
- Используйте колёсико мыши или щипок двумя пальцами для масштабирования (или кнопки +/-).

Редактирование объектов:
- Кликните на объект на поле, чтобы выбрать его. Панель справа/снизу покажет его свойства.
- Изменяйте JSON - изменения сразу отобразятся на карте.
- Если кликнуть по пустому месту, то можно редактировать JSON всей комнаты целиком!
Горячие клавиши:
• Delete / Backspace: Удалить объект.
• Стрелки: Перемещение объекта.
• + / -: Увеличить/Уменьшить размер.
• Ctrl+C, Ctrl+V, Ctrl+X: Копировать/Вставить/Вырезать.
• Ctrl+Z / Ctrl+Shift+Z: Отменить/Повторить изменения.
• Esc: Снять выделение / Выйти из Тест-режима.

Испытание (Тест):
- Нажмите "Испытать" чтобы "войти" в комнату в роли игрока.
- Появится джойстик для перемещения (мобильный/мышь) либо используйте WASD на клавиатуре.
- Зажав Shift или дважды кликнув по джойстику, вы ускоритесь.
- В этом режиме можно отключить/включить отображение границ коллайдеров.

Структура объекта (Для Дизайнеров):
Вы можете редактировать любые поля у объекта. Вот что они значат:
• id: Уникальное имя объекта.
• x, y, z: Позиция в чанках. z отвечает за слой (чем больше, тем выше/поверх остальных).
• color: Запасной цвет (HEX), если картинка не загрузится (например, "#ff00ff").
• sprite: Словарь с размерами (w, h) и ссылкой на картинку (url). Обратите внимание: размеры w и h задаются в чанках.
• collision: Словарь с коллайдером. Включает w, h (ширина/высота) и смещение от верхнего левого угла ox, oy. Если коллайдер меньше спрайта — получится проходимая зона за / перед объектом! Внимание: если w: 0 или h: 0, коллайдер полностью отключен.
• effects: Массив эффектов/триггеров.
- type: "active" или "passive"
- radius: Зона срабатывания (в чанках)
- action: Тип действия ("can_interact", "highlight")
- dialogue: Массив фраз NPC/Объекта
- npc_name: Имя говорящего.
Кнопка "Максимальный JSON" автоматически заполнит эти поля для выделенного объекта.

Ваша комната и добавленные извне спрайты автоматически сохраняется в браузере. Вы можете скопировать или скачать JSON для экспорта в игру.
